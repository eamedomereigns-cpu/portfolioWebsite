## About This Portfolio

This is my personal portfolio website showcasing my projects, skills, and contact information.  
It is designed to be clean, responsive, and recruiter-friendly.

---

## Features

- Responsive navigation bar with mobile menu toggle
- Project showcase cards
- Contact form with EmailJS integration (send me emails directly)
- Version tagging for easy maintenance
- Fully responsive design
- Built using **HTML**, **CSS**, and **JavaScript**

---

## Project Structure

- /index.html # Main portfolio page
- /style.css # Styling for portfolio
- /script.js # JavaScript for menu toggle, contact form, and animations
- /projects/ # Folder for project screenshots or demo files
- /assets/ # Images, icons, and other media
